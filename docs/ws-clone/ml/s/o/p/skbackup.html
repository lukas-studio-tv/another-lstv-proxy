About

SketchwareBackups Manager is tool developed by LSTV. Its an mobile app for Sketchware.
In SketchwareBackupManager, you can create an backup of all projects, collections etc. And then in backup manager easily restore/delete your created backups. If you restore backup, all data from backup will be transfered into Sketchware.
Example; something with your project goes wrong, like, you delete your project or mess something up. If you created backup before, You can restore everything!
