RSFW Docs

Wellcome to the docs, place to learn how to use RSFW.

Made with QM

Setup

To use RSFW, simply add this CDN into your HTML head:
<script src="https://lstv.ml/cdn/static/js/l/scratchfw_rsfw/0.1/min.js"></script>


Load project

First, you have to load the project.json file. (The project file.)
To get the file from your project, download the .sb3, extract it (unzip), and
open the project.json file. Copy what's inside and paste it here.
scratchOpenProject(jsonstring);

This automatically selects stage.
More onto selecting:

Selecting sprites

To modify sprites, you have to select the sprite to edit. Stage is selected by default.

2 ways to select a sprite:
1. By its position
setActiveSpriteAt(position);
(For example, stage is usually 0)

2. By its name
setActiveSprite(name);


Usefull variables/functions

isValidJson(jsonstring); - Validates json code and returns true/false
scratchProject - Actual project JSON
scratchProjectOutput() - Minified JSON string - used to save back to sb3
scratchProjectOutputFormatted() - Formatted JSON string - used for example to edit the direct JSON
scratchCurrentSprite - Gets current sprite's name
scratchCurrentSpriteLocation - Gets current sprite's position
scratchSpriteBlockAmount() - Gets current sprite's block count

Block commands


newBlock(opcode, inputs, fields, x, y, id);
(id is optional - if not specified, random ID will be used.)
Creates new block, not attached to anything, at a certain position.
