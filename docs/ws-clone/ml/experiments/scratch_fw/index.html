Welcome to the development verzion of Scartch FW.
About: This is a new project for automating the Scartch project editing outside scratch.

Tools:
- RSFW (Tool written primarily in JS. Allows to automate and edit projects with code. )
- S3TB (Some kind of code-based Scratch UI - Write code in text, then export to sb3.)
- RSBuilder (Automatic packager for sb3. Just open project/template, edit in editor with folder support, save and you have your custom sb3 with a click of a button.)
